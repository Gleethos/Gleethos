Voting Contract for the U.A.S - Technikum :

In the third semester of my undergrad studies I joined a Team working on an Ethereum smart contract intended to enable trust-less voting for my University.
We used solidity to write the contract and then built an interactive web form which connects to MetaMask (a browser plugin for blockchain account management) and automatically enables authorized parties to dispatch valid votes to a given blockchain.
