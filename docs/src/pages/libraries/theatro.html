A plotting library :

Theatro is small work in progress plotting library intended to to support the most basic types of plots in an intuitive way.

It is my second library besides Neureka, a tensor library.
I plan on integrating this library tightly with Neureka to allow for more use cases and to act as a light weight visual support tool.

Currently there are only 2 types of plots supported,
2D-grid plots and node-graph plotting.


This graph is a simple plot of points created from a sinus function.
The following code produced this grid plot : 


                Gridded g = new Gridded()
                g.setData(
                    [
                        (0..100).collect { (it - 50) / 10 } as double[],
                        (0..100).collect {Math.sin((it - 50)/10)} as double[]
                    ] as double[][]
                )
                

                Gridded g = new Gridded();
                g.setData(
                    new double[][]{
                        new double[]{...},
                        new double[]{...}
                    }
                )

The following example shows the plotting of a node-graph given a simple nested list/array structure...
The nodes can be dragged and dropped easily and the entire surface is navigable.
This graph has been created by the following code : 


                NodeWindow gb = new NodeWindow([
                    ["1", "2", "3"],
                    ["6", "4"],
                    ["8"]
                ])
                

                NodeWindow gb = new NodeWindow(
                    new String[]{
                        new String[]{"1", "2", "3"},
                        new String[]{"6", "4"},
                        new String[]{"8"}
                    }
                )

The library is in an extreme early phase of development.
However it already contains an extremely powerful base-layer used for extension and implementation of plots.
Drawing plots is as simple as passing a plotting lambda to a navigable surface backed by Swing.
