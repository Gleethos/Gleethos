Youtube Sentiment Analysis

Take a look at this project over at github!
In this project me and my team combined data scraping, analysis as well as machine learning in order to extract value from the collected data.
It started as part of a data science and big data class in the 5th semester of my undergrad studies at the U.A.S Technikum.
Me and my team partner wanted to collect data directly from the most widely used video sharing platform, namely: Youtube!

The most prominent technologies used:
- Pyppeteer for data scraping
- Neo4J for data storage
- Spark for data processing
- Sklearn and Matplotlib for analysis
- PyTorch for sentiment prediction
